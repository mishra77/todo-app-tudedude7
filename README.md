- Add new tasks  
- Edit existing tasks  
- Delete tasks  
- Filter tasks by priority  
- Simple and neat UI with EJS templates

## 🛠️ Technologies Used
- Node.js
- Express.js
- EJS (Embedded JavaScript templates)
- HTML, CSS
